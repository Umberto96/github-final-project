# Calcolatore di Interesse Semplice

## Descrizione
Questo progetto è un calcolatore di interesse semplice che permette di calcolare l'interesse guadagnato su un investimento.

## Formula
L'interesse semplice viene calcolato usando la formula:

**I = P × R × T**

Dove:
- **I** = Interesse
- **P** = Capitale principale (importo iniziale)
- **R** = Tasso di interesse annuale (in percentuale)
- **T** = Tempo (in anni)

## Esempio
Se investi €1000 con un tasso di interesse del 5% per 3 anni:
- P = 1000
- R = 5% = 0.05
- T = 3

Interesse = 1000 × 0.05 × 3 = €150

## Come usare
1. Inserisci il capitale principale
2. Inserisci il tasso di interesse annuale
3. Inserisci il periodo di tempo in anni
4. Calcola l'interesse
